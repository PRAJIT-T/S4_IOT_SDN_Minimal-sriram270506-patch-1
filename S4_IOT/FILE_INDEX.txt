├─ Purpose: Receives metrics, detects attacks, makes decisions

└─ Port: 9000

├─ Purpose: Simulates WiFi AP with 2 virtual nodes
├─ Ports: Listens on 9001 (UDP), sends to 9000

│  ├─ Simulates attack impact
│  ├─ Responds to controller commands
│  ├─ Switches channels
│  └─ Supports all 3 phases (baseline/attack/recovery)
├─ Run: python3 ap_agent.py
└─ Port: 9001

│  ├─ Coordinates with phases
│  └─ Sends metrics to controller
├─ Run: python3 monitor_agent.py
└─ Port: 9000 (outbound)

dashboard.py
