❓ "I have 2 virtual nodes right.... what's the dashboard.py for?"

✅ ANSWER:
   Dashboard.py is a web-based visualization tool:
   - Displays real-time metrics from controller, AP, and monitor agents
   - Shows throughput, RSSI, packet rate in live charts
   - Displays network status and jammer detection alerts
   - Helps visualize the attack impact visually
   - Access at: http://localhost:5000
   
   The 2 virtual nodes are simulated as connected clients in the WiFi AP
   that send and receive traffic to each other.

❓ "Create an attacker.py that comes into the AP and jams (pseudo 
    jamming), controller should print attack and isolate that IP. 
    I want this sort of demonstration"

✅ ANSWER:
   COMPLETELY IMPLEMENTED! Here's what you get:
   
   📄 attacker.py
   └─ Pseudo jamming simulator with:
      ✓ Targets specific WiFi channels (2, 6, or 11)
      ✓ Floods AP with fake jamming packets (100 packets/batch)
      ✓ Simulates realistic attack impact:
        • Packet rate: 300 pps → 8000+ pps (25x increase)
        • RSSI: -45 dBm → -70 dBm (drops 25 dBm)
        • Throughput: 4.5 Mbps → 0.9 Mbps (80% drop)
        • Packet loss: <1% → 80% (80x increase)
      ✓ Spoofs attacker ID (ATTACKER_XXXX)
      ✓ Sends jamming packets visible to controller
      ✓ Listens for isolation notice
      ✓ Stops automatically when isolated or time expires
   
   Controller Response:
   └─ Automatic detection and isolation:
      ✓ Detects jamming via 3-metric threshold:
        IF (packet_rate > 5000 AND rssi < -60 AND throughput < 2):
            ATTACK_DETECTED = True
      ✓ Prints attack alert:
        [Controller] 🔴 JAM PACKET DETECTED!
        [Controller] 🚨 JAMMING ATTACK DETECTED ON CHANNEL 6!
      ✓ Identifies attacker by IP:
        Attacker ID: ATTACKER_5432
        Source IP: 127.0.0.1
      ✓ Isolates the attacker:
        [Controller] 🔒 ISOLATING ATTACKER!
        Action: BLOCKING IP 127.0.0.1
      ✓ Sends isolation notice to attacker:
        [Attacker] 🚨 DETECTED AND ISOLATED BY CONTROLLER!
      ✓ Switches to clean channel:
        Decision: Switch from Channel 6 → Channel 11
