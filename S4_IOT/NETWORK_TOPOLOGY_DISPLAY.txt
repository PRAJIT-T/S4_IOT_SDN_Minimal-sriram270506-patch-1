HOW IT WORKS
════════════

Flow:
  1. AP Agent creates virtual nodes with IPs/MACs
  2. AP Agent sends this in metrics message: 'virtual_nodes': {...}
  3. Controller receives virtual_nodes data
  4. Controller displays nodes with their details
  5. Metrics appear right below the nodes

This creates a CLEAR CONNECTION between:
  ✓ Network topology (AP + Nodes)
  ✓ Metrics (from those nodes)
  ✓ Attack impact (on those nodes)
