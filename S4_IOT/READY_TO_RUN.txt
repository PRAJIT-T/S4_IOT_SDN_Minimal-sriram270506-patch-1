TROUBLESHOOTING

Issue: "Address already in use"
  Solution: pkill -9 -f "sdn_controller"
  Then try again

Issue: Still seeing zero metrics
  Solution: Make sure you're using the updated controller
  Run: grep "throughput_mbps" sdn_controller_multi_laptop.py
  (Should return a match)

Issue: "Can't connect"
  Solution: Make sure controller starts FIRST
  Wait 2-3 seconds before starting other components
