TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

Problem: "ryu module not found"
Solution: Ryu takes time to install. Wait for pip to complete.

Problem: "Mininet import error"
Solution: Mininet requires Linux kernel. Won't run on Windows/Mac.

Problem: "Port 6633 already in use"
Solution: Kill previous process: sudo killall -9 python3 controller

Problem: "iperf3 command not found"
Solution: Installed via pip. Use absolute path or check PATH.
